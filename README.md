# LegendsHR Bot 🤖

Telegram бот для збору даних співробітників у Excel.

## Функції

- 🌍 **3 мови:** Українська, Русский, English
- 💳 **Змінити реквізити** — збір ім'я + TRC20 гаманець
- 👤 **Новий співробітник** — збір ім'я + робочий телеграм
- 📞 **Зміна контактів** — збір ім'я + робочий телеграм
- ✅ **Підтвердження** перед збереженням
- ◀️ **Кнопки "Назад"** на кожному кроці
- 📊 **Excel файл** з окремими листами для кожного типу даних

## Структура

```
legends-hr-bot/
├── bot.py          # Основний код бота
├── data.xlsx       # Excel з даними (створюється автоматично)
├── bot.log         # Логи
├── start.sh        # Скрипт запуску
├── stop.sh         # Скрипт зупинки
├── requirements.txt
└── venv/           # Python virtual environment
```

## Запуск

```bash
# Запустити бота
./start.sh

# Зупинити бота
./stop.sh

# Переглянути логи
tail -f bot.log
```

| Лист | Колонки |
|------|---------|
